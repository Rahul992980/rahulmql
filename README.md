# EMA Crossover Indicator

## Overview
This is a TradingView Pine Script indicator based on EMA crossover logic to identify buy and sell signals.

## Strategy Logic
- Buy when Fast EMA crosses above Slow EMA
- Sell when Fast EMA crosses below Slow EMA

## Features
- Clear buy/sell signals
- Works on multiple timeframes
- Simple and easy to use

## How to Use
1. Open TradingView
2. Paste the script into Pine Editor
3. Add to chart
